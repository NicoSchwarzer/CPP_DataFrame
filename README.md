# CPP_DataFrame

A simple C++ project that mimics some functionalities of Python's **pandas DataFrame**.  
It supports columns with different data types (`int`, `double`, `string`, `bool`) and can read data from CSV files.

---

## Features

- Add columns dynamically with flexible data types using templates.
- Automatically detect cell types when reading CSV.
- Get the shape of the DataFrame (`rows x columns`).
- Retrieve column names and individual columns.
- Print and inspect your DataFrame content.
